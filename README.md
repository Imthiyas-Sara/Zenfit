# Zenfit

Zenfit is a fitness website developed as part of the **Emerging Technologies and Frameworks** module in Year 1, Semester 2.  
The project is designed to explore **Bootstrap frameworks**, learning how components like buttons, navigation bars, and grids work in a responsive web environment.

## Project Overview
Zenfit provides information and guidance on fitness, nutrition, workouts, and community engagement. The website contains the following pages:

- Home page by I. Sara  
- Nutrition Page by M.S.F Zikra 
- Workout Page by V. Kirushanthan  
- About Page  by S. Mahisha 
- Community  by F.Z. Insarf

The project uses **Bootstrap** to create a responsive, modern, and interactive web interface.


## Technologies Used
- **HTML5 & CSS** – Structure and styling  
- **Bootstrap** – Responsive layout, buttons, forms, and grid system
